Silent Movie App API Documentation

The Silent Movie App API provides access to information about silent movies, directors, and genres. Users can sign up, update their information, and manage their favorite movies list.

Endpoints

Method Endpoint Description Request Body Response
GET /movies Get all movies N/A An array of movie objects including title, description, genre, director, image URL, and featured status
GET /movies/:title Get a specific movie by title N/A A single movie object with details (title, description, genre, director, image URL, featured status)
GET /movies/genre/:genreName Get genre by name N/A A genre object with name and description
GET /movies/directors/:directorName Get director by name N/A A director object with name, bio, birth year, and death year
POST /users Create a new user JSON object with Username, Password, Email, Birthday JSON object of the created user
PUT /users/:Username Update a user's information JSON object with updated user info JSON object of the updated user
POST /users/:Username/movies/:MovieID Add a movie to a user's favorites N/A Updated user object with the new favorite movie
DELETE /users/:Username/movies/:MovieID Remove a movie from a user's favorites N/A Updated user object without the removed movie
DELETE /users/:Username Deregister a user N/A A success message indicating the user was deleted
