Silent Movie App API Documentation

The Silent Movie App API provides access to information about silent movies, directors, and genres. Users can sign up, update their information, and manage their favorite movies list.

Endpoints

Method Endpoint Description
GET /movies Get all movies
GET /movies/:title Get a specific movie by title
GET /movies/genre/:genreName Get genre details by name
GET /movies/directors/:directorName Get director details by name
POST /users Create a new user
PUT /users/:id Update a user's information
POST /users/:id/:movieTitle Add a movie to a user's favorites
DELETE /users/:id/:movieTitle Remove a movie from a user's favorites
