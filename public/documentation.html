Silent Movie App API Documentation

The Silent Movie App API provides access to information about silent movies, directors, and genres. Users can sign up, update their information, and manage their favorite movies list.

Endpoints

Method Endpoint Description
GET /movies Get all movies
GET /movies/:id Get a specific movie by ID
GET /movies/genres/:genreName Get movies by genre name
GET /movies/directors/:directorName Get movies by director name
POST /users Create a new user
PUT /users/:Username Update a user's information
POST /users/:Username/movies/:MovieID Add a movie to a user's favorites
DELETE /users/:Username/movies/:MovieID Remove a movie from a user's favorites
DELETE /users/:Username Deregister a user
